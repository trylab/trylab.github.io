旧电脑绕过 TPM/CPU 限制安装 Windows 11

Posted on 2024-05-31 | In Default

不依赖任何第三方工具，不修改设置，让旧电脑绕过 Windows 11 的 TPM 和 CPU 检测限制，直接升级 Windows 11。

0x01. 安装步骤

需要下载两个 ISO 镜像，一个是 Windows 10 的，一个是 Windows 11 的，把两个镜像解压，然后把 Windows 11 下的 sources\install.wim 替换 Windows 10 对应的文件即可。

如果是本机升级安装，那么直接通过解压、替换后的 Windows 10 操作即可。

如果是制作安装镜像，那么需要类似 UltraISO 之类的工具，替换 ISO 镜像文件中的 sources\install.wim。

优点：安全可靠，不依赖第三方 Patch 工具。

0x02. 优化设置

  1. 如果是覆盖安装，需要备份好 C 盘中的个人数据
  2. 备份 Office 激活文件 C:\Windows\System32\spp\store\2.0，升级后覆盖即可
  3. 绕过安装过程中的联网限制：在 OOBE 期间按下 Shift + F10，然后输入 oobe\bypassnro
  4. 磁盘属性中不再有磁盘清理入口，可以在开始菜单中搜索
  5. 锁屏界面中的应用信息（股票、天气等），可以在个性化设置中关闭
  6. 在应用 - 高级应用设置 - 应用执行别名中，关闭 App Installer 对 python.exe 和 python3.exe 的关联
  7. 禁用 Windows Search 服务
